#descripcion
"Diversificar tus ahorros debería ser tan simple como un clic. 
Koa es la fintech que tokeniza el precio del cacao para ofrecerte una alternativa real frente a la inflación. 
Cambia tu dinero por tokens de cacao en segundos, con total transparencia y liquidez inmediata. Es tecnología financiera con sabor a activo real."

#lenguajes utilizados
App Móvil:	Flutter
Backend:	Node.js + TypeScript
Base de Datos:	PostgreSQL
Servidor / Hosting	AWS (EC2 o App Runner)
Precio Cacao	API de Commodities: (Zyla Labs)
